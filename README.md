# Lexer Project
Lexer for a made up language RAT18S
Compile through the terminal using the command  
`g++ -std=c++17 lexer.cpp -o lexer`  
and run using the command  
`./lexer`  
This will result in a text file called `output.txt` that lists all the tokens and lexemes.
